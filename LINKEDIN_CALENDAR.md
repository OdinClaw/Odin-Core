**Type**: Tutorial / How-To Post
**Title**: "3 Skills You Need to Build AI Agents (and how to learn them fast)"

**Content Outline**:
1. **Python basics** (you don't need to be a coding wizard)
   - Loop through data, write functions, work with APIs
   - Free resource: Codecademy or FreeCodeCamp

2. **API thinking** (how to connect systems)
   - Understand request/response, JSON, authentication
   - Practice with a free API (weather, news)

3. **Problem decomposition** (the AI part)
   - Break big problems into small steps
   - Each step = one AI task or rule-based logic
   - Example: Claims routing = [extract info] → [categorize] → [assign]
